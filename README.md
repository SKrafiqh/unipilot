The Ultimate OS for Student Success. Built with React, Vite, and Firebase.

## Features

- **Notes & Exam Prep**: Curated study materials for engineering subjects.
- **Career Roadmaps**: Step-by-step guides for Internships and Placements with tracking.
- **Mini Projects**: Resume-ready project ideas with submission guidelines.
- **Premium Solutions**: Monetized access to source code and reports.
- **Dark Mode**: Fully responsive modern UI with light/dark themes.

## Tech Stack

- **Frontend**: React, Vite, TypeScript
- **Styling**: Tailwind CSS (v4)
- **Auth**: Firebase (Email Link / Passwordless)
- **Deployment**: Vercel

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Locally**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Deployment

This project is optimized for Vercel.

1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` to deploy.

## Project Structure

- `src/pages`: Main views (Landing, Dashboard, Modules)
- `src/components`: Reusable UI (ThemeToggle, PremiumLock)
- `src/data`: Static JSON content for Roadmaps, Notes, etc.
- `src/styles`: Tailwind CSS configuration
